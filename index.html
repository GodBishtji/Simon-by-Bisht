Press A Key to Start

How To Play

  1. Press any Key to Start. Simon will give you the first signal.Repeat the signal by pressing the same color button.
  2. Simon will then add one signal.Repeat these two signals by pressing the same color buttons, in order.
  3. Simon will again add one signal.Repeat these signals again and again.
  4. The game will be over as soon you can't follow the pattern.
  5. FOR MOBILE USERS ONLY Click on this area to open the keyboard to start the game.
